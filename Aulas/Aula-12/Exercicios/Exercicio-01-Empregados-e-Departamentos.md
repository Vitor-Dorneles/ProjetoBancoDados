# Exercício 01 — Empregados e departamentos

## Objetivo

Executar e interpretar uma sequência de scripts MySQL para compreender a criação de estruturas, inserção de dados e consultas com `SELECT`.

## Etapas

1. Execute [`01-criacao-empregados-departamentos.sql`](../Scripts/01-criacao-empregados-departamentos.sql) para criar o esquema e suas tabelas.
2. Execute [`02-insercao-empregados-departamentos.sql`](../Scripts/02-insercao-empregados-departamentos.sql) para popular o banco.
3. Execute e analise as consultas de [`03-consultas-empregados-departamentos.sql`](../Scripts/03-consultas-empregados-departamentos.sql).
4. Observe o uso de filtros, ordenação, valores distintos, operadores, agrupamento e funções agregadas.

Os enunciados específicos das consultas estão registrados como comentários no terceiro script.
