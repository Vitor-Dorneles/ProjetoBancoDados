# Exercício 04 — Cursos de pós-graduação e alunos

## Enunciado

Modele uma base acadêmica para registrar:

- cursos de **pós-graduação**;
- dados dos **cursos**, como nome e tipo;
- dados dos **alunos**, como nome, data de nascimento, sexo e histórico de créditos.

O arquivo preservado contém as entidades `PosGraduacao`, `Curso` e `Aluno`, mas não apresenta relacionamentos finalizados entre elas. Portanto, o modelo deve ser considerado uma tentativa ou versão incompleta do exercício.
