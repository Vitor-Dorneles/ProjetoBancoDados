# Exercício 03 — Controle de Manutenção de uma Oficina Mecânica

## Enunciado

Uma oficina quer organizar seus serviços. Um **cliente**, identificado por nome, telefone e CPF, pode possuir vários **veículos**, com placa, modelo e marca, mas cada veículo pertence a apenas um cliente.

Quando um veículo apresenta defeito, ele passa por uma **ordem de serviço**, que registra a data de entrada e o valor total. Uma ordem de serviço é sempre referente a um único veículo.

Além disso, uma ordem de serviço pode envolver vários **mecânicos**, identificados por nome, especialidade e número de registro, e um mecânico pode trabalhar em diversas ordens de serviço ao longo do dia.
