# Banco de Dados

é um conjunto de dados integrados que tem por

objetivo atender a uma comunidade de

usuários

- Compartilhamento de dados
- aborda a importância dos dados de um produto por exemplo, em uma empresa de vendas
- aborda também o problema da estruturação de **sistemas** **isolados** que causam a **redundância de dados**

### REDUNDANCIA DE DADOS

- Controlada - onde o sistema sabe desses diversos dados, usa para melhorar o desempenho
- Não controlada - é o usuario que controla e não o software
		Mesma informação sendo digitada diversas vezes, inconsistencia de informação,
A solução para a redundancia não controlada é o compartilhamento de dados

## Sistema de gerencia de banco (SGBD)

construidos para atender a complexidade dos diferentes requisitos de um sistema

**Vantagens do sgbd**

- manutenção mais simples, pois temos uma separação clara de funções
- Maior compreensão
- Reutilização de funções

#### Tipos de SGBD

**RELACIONAL:**

- domina o mercado atual
**ORIENTADO A OBJETOS**
**Não Relacional**
