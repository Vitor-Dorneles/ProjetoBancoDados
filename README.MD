# 🗄️ Projeto de Banco de Dados

Este repositório contém estudos, exercícios e diagramas sobre a disciplina de **Projeto de Banco de Dados**, desenvolvidos no 3º semestre da graduação em **Ciência da Computação**. O foco é praticar o levantamento de requisitos e a construção de modelos conceituais e lógicos.

---

## 🏛️ Instituição
- **Universidade:** Universidade Franciscana (UFN)
- **Curso:** Ciência da Computação

## 📖 Bibliografia Base

O conteúdo e os exercícios práticos deste repositório são fundamentados na principal referência bibliográfica da disciplina:

> HEUSER, Carlos Alberto. **Projeto de Banco de Dados**. 6. ed. Porto Alegre: Bookman, 2009.

## 🛠️ Tecnologias e Ferramentas
- **IDE:** VS Code
- **Controle de Versão:** Git & GitHub
- **Formatos:** Markdown (`.md`) para documentação das regras e `.png` para a exportação dos diagramas.
