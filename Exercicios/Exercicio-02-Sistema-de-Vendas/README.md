# Exercício 02 — Sistema de vendas

## Enunciado

Modele um sistema de vendas que controle:

- vendedores, identificados por código e nome;
- clientes, identificados por código ou CPF, com nome e endereço;
- produtos, identificados por código, com nome, tipo e localização na prateleira;
- vendas, com código, cliente, vendedor, valor, data e produtos comercializados.

Considere que um vendedor pode realizar várias vendas, um cliente pode participar de várias vendas e uma venda deve envolver um ou mais produtos.
